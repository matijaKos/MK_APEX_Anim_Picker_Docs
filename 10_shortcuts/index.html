    Keyboard & Mouse Shortcuts

    Navigation

    Input Action
    Scroll Wheel Zoom in / out
    Alt + Scroll Zoom in / out (same as scroll)
    Middle Click + Drag Pan
    Alt + Middle Click + Drag Pan (same as middle click)
    H Fit all buttons in view
    G Fit selected buttons in view (falls back to fit-all if nothing selected)

    Selection (non-edit mode)

    Input Action
    Left Click (button) Select only that button, deselect all others
    Shift + Left Click (button) Add button to selection
    Ctrl + Left Click (button) Deselect that button
    Alt + Left Click (button) Select button + its mirror controls in APEX
    Left Drag (empty space) Replace selection — rubber-band (blue)
    Shift + Left Drag (empty space) Add to selection — rubber-band (orange)
    Ctrl + Left Drag (empty space) Remove from selection — rubber-band (red)
    Left Click (empty space) Deselect all (picker + APEX viewport)

    Edit Mode

    Input Action
    Left Click (button) Select only that button
    Shift + Left Click (button) Add to selection
    Ctrl + Left Click (button) Deselect that button
    Left Drag (button) Move selected buttons
    Left Drag (resize handle) Resize button
    Double Click (button) Rename button
    Right Click (button) Context menu
    Delete / Backspace Delete selected buttons
    Ctrl + C Copy selected buttons
    Ctrl + V Paste buttons

    Zone Edit Mode

    Input Action
    Left Drag (empty canvas) Draw a new zone
    Left Drag (zone) Move zone
    Left Drag (zone edge/corner) Resize zone
    Left Click (zone) Select zone
    Delete / Backspace Delete selected zone
    Left Drag (zone square in HUD) Reorder zones
    Left Click (zone square in HUD) Zoom canvas to that zone

    Button Right-Click Menu (select mode)

    Action Description
    Reassign to Selection Overwrite button's controls with current APEX viewport selection
    Add Selection to Button Append current APEX viewport selection to button's controls
    Edit Controls Open Selection Manager dialog
